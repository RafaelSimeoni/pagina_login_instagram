# Cópia da página de login do Instagram
### Tecnologias utilizadas:

 - HTML5
 - CSS3

### Descrição:
A cópia da página de login do Instagram é o meu primeiro projeto compartilhado no GitHub. Trata-se de uma atividade desenvolvida no curso "HTML Web Developer", oferecido pela Digital Innovation One. A princípio optei por desenvolver a página de login do Instagram sem o auxílio das aulas de criação desse projeto, para melhorar meus conhecimentos nas tecnologias envolvidas. Para isso, utilizei o modelo de layout CSS flexbox, e media querys para tornar a página responsiva. Além disso, esse projeto me ajudou a melhor fixar os conhecimentos em git e na plataforma GitHub.
